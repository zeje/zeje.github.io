# 维也纳酒店集团

维也纳酒店有限公司创始于1993年，是中国中档商务连锁酒店知名企业，连续七年蝉联中国连锁酒店中端品牌规模第一（出自盈蝶咨询2013-2019中国酒店连锁发展与投资报告）。2016年，锦江股份战略投资维也纳酒店有限公司，维也纳酒店有限公司成为全球前三强酒店集团的成员之一。
