# 深圳市中安信业创业投资有限公司

深圳市中安信业创业投资有限公司（中安信业）成立于2003年10月，是一家小额信贷技术服务企业，是世界银行国际金融公司（IFC）的参股企业，也是中国小额贷款公司协会副会长单位及深圳市小额贷款行业协会会长单位。
