# 中山益达服装有限公司

中山益达服装有限公司是晶苑旗下一家大型港资牛仔服装制造企业。创办于2003年12月，投资总额近5亿港元，占地面积约近13万平方米，设有现代化标准生产大楼、智能式办公楼，以及设施齐全的员工宿舍楼，厂区绿化面积占总面积的30%，被誉花园式工厂。
