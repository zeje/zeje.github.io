# 江门市中港宝田摩托车实业有限公司

江门市中港宝田摩托车实业有限公司成立于一九九四年。位于珠江三角洲城市江门。江门是目前中国最大型的摩托车生产基地之一。公司主要出口“宝田”牌系列49CC、125CC、150CC三大系列摩托车及摩托车发动机。
